✅ События календаря (естественный язык)
   "Создай встречу на завтра в 10:00"
   "Покажи события на сегодня"
   "Что у меня в пятницу?"

✅ Задачи Google Tasks (команды)
   /create_task Купить молоко | завтра
   /tasks - посмотреть все задачи
   /complete_task 1 - отметить выполненной

✅ Комбинированный просмотр
   /today_all - события + задачи на сегодня
   /week_all - события + задачи на неделю

✅ Поддержка русского языка
   сегодня, завтра, вчера, позавчера
   понедельник, вторник, среда...
   15.01, 15 января, 20.12.2025
   10:00, 10-00, 10ч
